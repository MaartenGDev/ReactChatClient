# ReactChatClient
![Interface Preview](https://github.com/MaartenGDev/ReactChatClient/blob/master/public/images/interface_dark.PNG)

## Install
- git clone git@github.com:MaartenGDev/ReactChatClient.git
- cd ReactChatClient
- npm install
- node index.js
- go to: [localhost:3001](localhost:3001)
- Enjoy
